# Simple-Interest-Calculator

This project is a **Simple Interest Calculator** that allows users to calculate the interest generated from an initial amount of money, an interest rate, and a period of time.

## 📌 Description

The application allows the user to enter:

* Initial capital (principal)
* Interest rate
* Time period

Using these values, the program calculates the **simple interest** using the formula:

Simple Interest = Principal × Rate × Time
